# 豆包PPT定制流程

## 概述

本流程描述如何使用豆包AI根据需求定制PPT，包含从需求准备到最终导出的完整步骤。

## 第一步：准备需求框架

在向豆包输入需求前，先准备好以下关键信息：

| 要素 | 说明 |
|------|------|
| **主题** | PPT的核心主题是什么 |
| **受众** | 目标受众是谁（客户、内部团队、政府官员等） |
| **用途** | 项目推介、培训、汇报还是展示 |
| **结构** | 需要包含哪些章节 |
| **风格** | 商务简约、科技感、学术严谨等 |
| **字体格式** | 是否有品牌字体要求 |
| **页数** | 大致页数要求 |
| **特殊需求** | 是否需要图表、数据可视化等 |

### 示例需求模板

> 帮我制作一个PPT，主题是"AI智慧农业应用方案"，目标受众是农业企业客户，用途是项目推介。需包含以下章节：
> - 行业背景与痛点
> - AI技术在农业中的应用场景
> - 成功案例
> - 合作模式与预算
>
> 风格要求商务简约风，页数12页左右，在"应用场景"部分用雷达图展示不同技术的适配度，每页添加页码，重点突出成功案例的效果数据。

## 第二步：输入内容生成PPT

根据素材情况选择合适的方式：

### 方式1：纯文字描述
直接复制准备好的需求模板，粘贴到对话框。可补充细节调整，如"增加一页数据总结页"。

### 方式2：上传文档
若已有 Word/PDF/TXT/Excel 格式的文字稿，可直接上传文档，并补充指令：
> "根据此文档内容生成PPT，保留核心数据，简化文字，采用科技感风格"

### 方式3：粘贴大纲
若已手动整理好PPT大纲，可直接粘贴大纲，补充风格、页数等要求，让豆包基于大纲填充内容和设计排版。

### 方式4：多文件合并（高级版）
若素材分散在多个文件中，可上传多个文件，指令豆包"合并文件内容，生成符合要求的PPT"。

> **提示**：输入指令后，可补充"一键优化"需求，让豆包自动调整布局、补充细节（如添加引用来源）。

## 第三步：迭代修改

若预览后发现内容偏离需求或不符合预期，**无需重新生成**，直接在对话框输入修改指令进行多轮迭代（通常2-3轮即可）。

### 结构调整
- "将'合作模式'章节提前至第3页，与'成功案例'相邻"
- "删除'行业背景'中的XX内容，补充XX痛点"
- "将'成功案例'章节从2页扩展为3页，补充每个案例的效果数据"

### 内容调整
- "把'行业背景'页的文字精简，提取3个核心痛点，用项目符号展示"
- "替换'案例数据'中的2025年数据为2026年最新数据，用折线图展示年度增长"
- "添加一页关于解决方案的内容，包含3个具体措施"

### 风格调整
- "更换模板为'科技感深蓝风'，重新生成排版"
- "将主色调改为公司VI色，统一字体为思源黑体"
- "删除所有卡通元素，调整为严谨学术风格"

### 高级技巧：分批生成法
对于20页以上的复杂PPT：
1. 先让豆包生成大纲，确认章节逻辑
2. 再分章节指令生成详细内容（如"先生成1-5页（背景+现状），确认后再生成后续页面"）
3. 避免一次性生成后大幅修改

## 第四步：复核检验与细节调整

### 文本修改
- 点击任意文本框，修改文字内容
- 调整字体大小、加粗重点语句
- 删除冗余内容
- 确保每页文字简洁、重点突出
- **原则**：每页一个核心观点，文字不超过6行
- 修正错别字和语法错误

### 素材替换与添加
| 操作 | 说明 |
|------|------|
| **替换** | 将豆包自动生成的占位图，替换为提前准备的公司LOGO、产品图片、案例图片等，确保图片清晰、与内容相关 |
| **添加** | 插入自定义素材（如数据表格、流程图、图标），丰富PPT内容。如在数据页插入精准的Excel表格，在流程页插入流程图 |

> **注意**：插入时注意图文排版均匀，避免遮挡文字

## 第五步：导出

根据提示点击导出按钮，下载最终PPT文件。
