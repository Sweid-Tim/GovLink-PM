# GovLink 签证办理综合指南

## 客户信息脱敏与外发口径

签证材料内部整理、正式邀请函、正式派遣函、报批材料和签证正式材料，按业务要求使用客户正式名称。凡是生成给代办机构、保险供应商、票代、酒店、外方邀请单位或其他第三方的前期咨询口径，必须按最小必要原则脱敏；不得直接写政府客户、事业单位客户、国企客户全称，除非该信息是正式材料必填且已由项目负责人确认。外发咨询阶段使用“华东某政府团组”“江浙沪某国家级开发区”“江苏某国家级开发区”等模糊表述，并在末尾标注：`客户名称已脱敏，正式披露前需由项目负责人确认。`

## 概述

公务团组签证办理的标准流程，覆盖需求澄清、材料准备、各国签证要点。本文档整合了公司多年项目实践中积累的签证知识，适用于政府出访团组的签证办理需求。

---

## 一、签证需求澄清（第一步）

办理前必须确认以下信息：

| 序号 | 问题 | 说明 |
|:----:|------|------|
| 1 | 想要申请的国家 | 具体到国家名称 |
| 2 | 签证类型 | 旅游/商务/探亲访友等 |
| 3 | 预计出行时间 | 具体到XX月XX日-XX月XX日 |
| 4 | 行程计划 | 到访国家（是否还有其他国家） |
| 5 | 出行人数及关系 | 几个人，彼此关系 |
| 6 | 申请人户籍及常住地 | 户籍所在地、目前常住地 |
| 7 | 国外旅行记录 | 是否有过出境记录 |
| 8 | 拒签记录 | 是否有被任何国家拒签过 |
| 9 | 收入情况 | 是否稳定（在职/自营/退休/自由职业） |

## 二、各国签证基本信息

### 亚洲

#### 日本签证
- **类型**：单次/三年多次/五年多次
- **审核时间**：6个工作日
- **签证费**：代办社价格单次200元，多次390元（2026年4月起涨价）
- **服务费**：单次500元，多次800元
- **申请条件**：
  - 单次：近一年工资流水10万以上 / 纳税800元以上 / 10万存款证明+工资流水 / 全日制四年本科在读或毕业三年内
  - 三年多次：近一年工资流水20万以上 / 纳税1.2万元以上 / 50万存款证明+工资流水 / 近三年持有两次日本旅游个签
  - 五年多次：近一年工资流水50万以上 / 纳税7.8万元以上 / 100万存款证明+工资流水
- **材料清单**：[日本签证材料](https://brjv2csped.feishu.cn/wiki/GPXsw7H7liye7mkOWv8cgfBNn7c)

#### 韩国签证
- **类型**：单次/半年两次/五年多次/十年多次（电子签）
- **审核时间**：5-10个工作日
- **签证费**：单次280元，多次630元，签证中心服务费120元
- **服务费**：单次700元，5年多次1200元，10年多次1500元
- **五年多次条件**：芝麻信用分≥780 / 上海/杭州/宁波/南京/苏州户籍 / 全日制四年本科在读或毕业 / 房产200万以上 / 银联白金信用卡 / 月收入5000元等

## 三、签证办理核心原则

### 1. 材料完整性
- 确保所有要求的材料齐全
- 缺少关键材料可能导致拒签

### 2. 真实性
- 所有材料必须真实有效
- 严禁伪造、变造材料

### 3. 提前准备
- 至少提前1-2个月开始准备
- 预留充足时间应对突发情况

### 4. 行程明确
- 提供详细的行程计划
- 机票、酒店订单需与行程匹配

### 5. 资金证明
- 提供足够的资金证明
- 银行流水需真实、连续

### 6. 出入境记录
- 保持良好的出入境记录
- 无严重违规或滞留记录

## 四、签证办理流程

### 标准流程

```
需求澄清 → 材料准备 → 填写申请表 → 递交申请 → 缴纳费用 → 等待审核 → 领取护照
```

### 注意事项

- **护照有效期**：需在半年以上
- **照片规格**：符合各国签证照片要求
- **行程单**：需与机票酒店订单一致
- **保险**：申根签证需购买境外保险（30万保额以上）
- **公证认证**：部分材料需办理公证和海牙认证

## 五、政府团组特别说明

### 因公护照 vs 因私护照

| 类型 | 办理渠道 | 说明 |
|------|---------|------|
| 因公护照 | 外事办公室统一办理 | 政府团组走因公渠道 |
| 因私护照 | 委托旅行社或自行办理 | 可自行准备材料 |

### 部分国家免签政策（公务护照）

| 国家 | 公务护照免签 | 备注 |
|------|-------------|------|
| 南非 | 免签（≤30天） | 超过30天需办理居留手续 |
| 尼日利亚 | 免签（≤30天） | 依据中尼互免签证协定 |
| 摩洛哥 | 免签（≤90天） | 中国护照免签 |
