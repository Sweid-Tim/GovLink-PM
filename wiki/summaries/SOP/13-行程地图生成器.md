# Itinerary Map Maker - 行程地图生成器

## 技能描述

根据用户提供的行程点列表，生成交互式 HTML 地图。使用 Leaflet + OpenStreetMap，无需 API Key，开箱即用。

---

## 使用场景

当你需要：
- 制作旅行路线可视化地图
- 展示多城市/多景点行程顺序
- 生成可分享的交互式行程图

---

## 触发方式

直接调用，传入行程点列表：

```
/itinerary-map-maker
```

---

## 执行流程

### Step 1: 收集行程点

请用户提供行程点列表，格式要求：
- 每行一个城市/景点
- 或用 `->` 或 `-` 连接
- 可标注国家（用 `/` 分隔）
- 可添加备注（用 `:` 分隔）

**示例格式**：
```
日内瓦 / 瑞士
洛桑 / 瑞士
琉森 / 瑞士 -> 龙疆 / 瑞士
少女峰 / 瑞士 : 需乘火车
维罗纳 / 意大利
罗马 / 意大利
```

### Step 2: 生成 HTML 地图代码

使用以下 prompt 模板让 AI 生成代码：

```
请根据以下行程顺序，制作一个交互式 HTML 地图，要求如下：

1. **城市顺序**（按天或站点排列）：
[粘贴行程点列表]

2. **地图功能要求**：
   - 在地图上精确标注每个地点（使用真实地理坐标）
   - 按顺序绘制连续路线
   - 点击每个标记点时，弹出窗口显示：
     • 城市名称
     • 所属国家/地区
     • 在行程中的序号（如"1. 日内瓦"）
     • 可选备注（如"少女峰门户""薰衣草田"等，如有请提供）
   - 首站（第一个城市）用黄色圆点 + 黑色数字"1"高亮显示
   - 其他站点用统一颜色（如蓝色或红色）+ 白色序号标记

3. **技术要求**：
   - 使用 Leaflet + OpenStreetMap
   - 纯 HTML 文件，无需外部依赖（除 CDN 外）
   - 包含图例说明（首站标识、路线含义）
   - 支持缩放、拖动、点击交互
   - 代码可直接保存为 .html 文件并在浏览器中运行

4. **其他说明**：
   - 若有重复城市（如"维罗纳"出现两次），地图上只显示一个标记，但路线仍按完整顺序连接
   - 对模糊地名（如"AIX"）请自动识别为标准名称（如"Aix-en-Provence"）
   - 尽量使用权威地理坐标（优先选用市中心或代表性景点坐标）

请输出完整的 HTML 代码。
```

### Step 3: 保存为 HTML 文件（Mac 用户推荐方法）

Mac 系统直接用「文本编辑」保存，无需改后缀：

1. **打开文本编辑**
   - 按 `Command + 空格`，搜索「文本编辑」
   - 新建空白文稿

2. **粘贴代码**
   - 把生成的 HTML 代码全部复制粘贴进去

3. **设置纯文本格式（关键！）**
   - 顶部菜单栏 → `文本编辑` → `偏好设置`
   - 点开「格式」选项卡
   - 选择：**纯文本**
   - ⚠️ 不选纯文本会乱码！

4. **保存文件**
   - `Command + S` 或菜单 `文件` → `存储`
   - 文件名：`瑞士行程地图.html`（注意 .html 后缀）
   - 位置：桌面或其他想要的位置
   - 存完后直接双击即可用浏览器打开

---

**备选方案**：如果是命令行用户，也可以保存到指定目录：

**默认位置**：`wiki/travel-maps/`

**文件名格式**：`YYYYMMDD-行程路线.html`

**示例**：`20260512-欧洲之旅.html`

### Step 4: 输出结果

告知用户：
- 文件保存路径
- 直接用浏览器打开即可使用
- 可嵌入 Obsidian 或分享给他人

## 注意事项

### 常见问题

**Q: 打开 HTML 文件是代码而不是地图？**
- 原因：保存时没有选择「纯文本」格式
- 解决：重新打开 → 菜单 → `文本编辑` → `偏好设置` → `格式` → 选「纯文本」→ 重新保存

**Q: 双击文件用 Safari 打开一片空白？**
- 原因：可能是纯文本模式下保存的文件
- 解决：右键文件 → 「打开方式」→ 选择 Chrome/Firefox

**Q: 想存到其他位置？**
- 在保存对话框中选择目标文件夹即可
- 推荐：`wiki/travel-maps/` 目录下统一管理

### 技术限制
- 生成的 HTML 依赖 CDN，需联网使用
- 离线使用需内联 Leaflet 库（约 150KB）
- 如需嵌入 Obsidian，可使用 iframe 插件
