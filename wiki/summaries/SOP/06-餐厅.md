## 客户信息脱敏与外发口径

当餐厅方案中需要生成给餐厅、地接、导游、供应商或付款方的预订、询价、菜单确认口径时，必须先对客户身份脱敏。不得直接写政府客户、事业单位客户、国企客户全称；使用“华东某政府团组”“江浙沪某国家级开发区”“江苏某国家级开发区”等模糊表述。可描述人数、日期、餐标、包间、禁忌和服务要求，但不得暴露客户单位全称、领导姓名、团员名单和预算细节。正式合同、账单、付款文件如业务要求必须写明客户正式名称，应在项目负责人确认后使用正式名称。对外预订/询价版末尾标注：`客户名称已脱敏，正式披露前需由项目负责人确认。`


**⚠️ 每次输出餐厅方案时，必须严格按以下格式输出。不得更改字段顺序或省略任何字段。请务必确保所有信息（特别是营业状态和预订要求）真实可靠，严禁胡编乱造。**

输出格式如下：

| 餐厅名称    |     |
| ------- | --- |
| 餐厅类型与档次 |     |
| 餐厅简介    |     |
| 地址      |     |
| 菜品照片    |     |
| 营业时间    |     |
| 预定要求    |     |
| 是否有包厢   |     |
| 预定电话    |     |
| 人均消费参考  |     |
| 项目建议    |     |
| 信息来源    |     |


餐厅的推荐请参考：**米其林指南、Black Pearl（黑珍珠）、TripAdvisor（猫途鹰）、Google Maps评价、小红书（看真实返图）**等。请优先筛选评分在 4.0 以上或有特殊口碑的餐厅。

- **餐厅名称：**（中英文对照）
- **餐厅类型与档次：**（例如：法式正餐、当地特色私房菜、商务宴请、高空景观餐厅等）
- **餐厅简介：**（200字以内，重点描述餐厅特色、招牌菜及环境氛围）
- **地址：**（附上谷歌地图链接或经纬度，方便模拟动线）
- **菜品照片：**（至少包含3张：环境图、招牌菜图、菜单图）。如无法直接提供图片，则可以提供图片的链接。
- **营业时间：**（需注明是否有午休时间/Last Order时间，以及周几公休）
- **预订要求：**（是否需要提前预订？提前几天？是否需要预付定金或信用卡担保？）
- **人均消费参考：**（当地货币及折合人民币价格，是否包含服务费/税费）
- **项目建议：**（是否适合作为公务晚宴？是否有独立包间？是否有中文菜单或懂中文的服务员？是否有特殊的着装要求/Dress Code？）
- **信息来源：**（请注明参考网站或APP的链接）

### 💡 给小龙虾的补充操作建议：

1. **口味匹配：** 如果是长期出差，请在方案中穿插 1-2 家高品质的中餐厅（中式胃救星）。
2. **距离优先：** 餐厅的选择必须基于你当天下午最后一个公务点或入住酒店的**步行距离/车程 20 分钟内**。
3. **避雷指南：** 评价中如果出现多次“上菜极慢”、“环境嘈杂”或“对华人不友好”的情况，直接剔除。
4. **实时确认：** 最终选定前，必须通过 Google Maps 或官网确认该餐厅在行程当天是否被包场或因节假日歇业。
