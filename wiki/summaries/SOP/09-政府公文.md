## 客户信息脱敏与外发口径

政府公文、正式邀请函、报批材料、签证正式材料等政府官方沟通文件，按业务要求使用客户正式名称，不做模糊化处理。脱敏规则适用于正式发文前的询资源、询格式、询接待、询价和第三方协作沟通：向南京西联、地接社、境外机构、商协会、企业或供应商询问接待可能性时，不得直接写政府客户、事业单位客户、国企客户全称；使用“江苏某国家级开发区”“江浙沪某国家级开发区”“华东某政府产业部门”等模糊表述。对外咨询版末尾标注：`客户名称已脱敏，正式披露前需由项目负责人确认。`


**注意：**

> “公文排版是一项严谨的政治任务。排版错了，内容再好也会被认为专业度不足。请小龙虾们务必保存此手册，每次发文前对照勾选一遍！”。注意你一定以word文件格式发给我，word文件里的格式一定严格遵照以下格式要求。

# 中国党政机关公文格式标准执行手册（优化版）

## 一、 页面设置（决定公文的“精气神”）

|**项目**|**标准要求**|**Word设置提示**|
|---|---|---|
|**纸张尺寸**|A4纸（210mm × 297mm）|默认A4|
|**页边距**|上37mm，下35mm，左28mm，右26mm|页面布局 -> 自定义边距|
|**行距**|**固定值 28pt 或 29pt**|段落 -> 行距 -> 固定值|
|**排版密度**|每页22行，每行28个字|确保字间距与行间距的疏朗感|

## 二、 字体与字号标准（公文的“门面”）

| **要素**   | **字体**        | **字号** | **备注**                    |
| -------- | ------------- | ------ | ------------------------- |
| **公文标题** | **方正小标宋简体**   | **二号** | 居中，不加粗（字体本身偏厚），分行需对齐      |
| **发文字号** | 仿宋_GB2312     | 三号     | 如：国发〔2026〕5号（六角括号不可错）     |
| **主送机关** | 仿宋_GB2312     | 三号     | 左顶格，后跟全角冒号                |
| **正文内容** | **仿宋_GB2312** | **三号** | 每个自然段首行缩进2字符              |
| **附件说明** | 仿宋_GB2312     | 三号     | 正文下空一行，左空二字标注“附件：”        |
| **发文机关** | 仿宋_GB2312     | 三号     | 位于正文右下方，居中于日期上方           |
| **成文日期** | 仿宋_GB2312     | 三号     | 右空四字，阿拉伯数字标全，如：2026年4月14日 |

## 三、 标题层次规范（严格按此顺序，不得跳级）

- **第一层：一、**（**黑体**，三号，不加粗，末尾无标点）
    
- **第二层：（一）**（**楷体_GB2312**，三号，加粗，末尾无标点）
    
- **第三层：1.**（**仿宋_GB2312**，三号，加粗，**注意是实心圆点**）
    
- **第四层：（1）**（**仿宋_GB2312**，三号，不加粗）
    

## 四、 细节避雷（“小龙虾”最易出错点）

1. **标点符号：** 必须使用**全角标点**。尤其是括号，严禁使用半角 `( )`，必须使用全角 `（ ）`。
    
2. **数字规范：** 公文中的年份必须使用六角括号**〔 〕**，严禁使用方括号 `[ ]` 或中括号 `【 】`。
    
3. **禁止孤行：** 严禁出现“孤行压页”（即正文最后一行单独出现在第二页），至少要带两行正文过去。
    
4. **页码格式：** **4号半角宋体**，位于页脚居中。格式为：`- 1 -`。
    
5. **特定禁词：** 涉及“报批行程”时，严格执行之前定义的**禁用词汇表**（禁用：考察、参观、景点等）。
    

## 五、 给“小龙虾”的Word操作建议

1. **字体安装：** 电脑里必须安装 **“方正小标宋简体”** 和 **“仿宋_GB2312”**。
    
2. **网格设置：** * 页面设置 -> 文档网格 -> 网格 -> 指定行和字符网格。
    
    - 设置：每行28个，每页22行。这样系统会自动调整字间距。
        
3. **自动检查：** 导出PDF前，务必检查一遍“六角括号”和“层级序号”是否统一。
