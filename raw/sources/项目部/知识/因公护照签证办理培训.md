## 1. 因公护照申办指南（省级）

### 受理部门
**省政府外事办公室出国管理处**（南京）

### 办理依据
1. 出国任务件
   - 参考网站：https://wb.jiangsu.gov.cn/art/2024/8/30/art_352_2550385.html
2. 出国任务通知书

### 申请条件
全省各市外办，省各部、委、办、厅、局，省直属单位，大型企业集团，部分外贸企业，大专院校、科研机构以及经外交部批准授权的单位。

### 办理时限
一般情况下，护照 **三个工作日** 后可领取。

### 办理流程
1. **网上申报材料**
2. **交纸质材料到办照窗口**
3. **采集指纹** — 申请新护照的出国人员带身份证原件到窗口采集指纹（现役军人及申办通行证者除外）

### 独立申办权限单位
南京、南通、连云港市及苏州工业园区外办具备因公护照**独立申办权限**，按三市一区规定的程序申办，无需通过省级外办审批。

## 2. 因公护照申办材料清单

1. **出国任务批件**原件
   - 双跨团组提供：任务通知书、任务批件、任务确认件
2. **因公临时出国（境）人员备案表**原件
3. **合格有效的邀请函**电子版/复印件
4. **因公出国人员所在单位按干部管理权限，向上级报批的出国请示**原件
5. **出国人员居民身份证**（军官证、警官证）复印件
6. **照片** — 正面、免冠、白底 2 寸、服装整齐的软光纸近期照片
7. **申请出国护照签证事项表**
8. **因公电子护照申请表**

## 3. 因公护照申办指南（南通地区）

> 南通地区的护照办理可与请示报批同步进行。

### 流程
1. 携带身份证原件前往**市外办**采集指纹
2. 拍照
3. 网上申报材料
4. 递交纸质材料至市外办

### 纸质材料清单
| # | 材料名称 | 说明 |
|:-:|:---|:---|
| 1 | 《申请出国护照签证事项表》 | 市外办领取 |
| 2 | 《因公电子护照申请表》 | 市外办领取 |
| 3 | 《申办出国境手续历史情况登记表》 | 市外办领取 |
| 4 | 《护照填写卡片》 | 市外办领取 |
| 5 | 《因公临时出国人员备案表》原件 | 由团组成员所在单位填写并**加盖公章** |

> **注意**：以上护照办理材料可在递交签证材料前/当天交到市外办。

### 所需时间及特殊情况
- 一般流程：填报请示 + 录完指纹后，**2个工作日**出护照号，**4个工作日**护照原件到达市外办
- 可利用这段时间向团组成员收集办理护照和签证所需信息及材料
- **紧急情况**：如出行时间紧张但邀请函未确定、无法上报请示时，可向市外办说明计划出行时间及国家，由市外办领导确认是否可以在系统**容缺提交**护照申请

## 4. 因公签证申办指南

### 前期准备
- 参考网站：
  - **江苏省签证服务中心** — 各国签证要求
  - **江苏省外事办** — 最新签证信息
- 在等待护照办结过程中，提前查看申办国签证材料清单及注意事项
- 中心网站上没有的国家，通常为**免签或落地签**国家

### 签证预约
| 类型 | 预约方式 |
|:---|:---|
| 美国、澳新、丹麦 | 可自行预约时间递交材料 |
| 其他需预约的国家（如申根） | 需由**市外办与省外办**预约 |
| 申根国家指纹采集 | 无论是否有过申根指纹，都需预约递交名额；签证材料要在预约时间之前进领馆 |

### 签证费用
- 省外办结算发送账单至市外办 → 市外办通知团组付费
- 费用包含：
  - 签证费（线上缴费国家除外：如丹麦、英国等）
  - 省外办服务费
  - 护照往返快递费用

### 签证材料递交
- 由 **团组专管员 / 团组代表 / 组团单位代表** 将材料递交至市外办
- 第三方**仅可协助，不可单独递交**

## 5. 签证材料注意事项 — 日本

### 日本签证邀请函（固定模板）
由以下四份文件组成：

| 文件 | 盖章要求 |
|:---|:---|
| 招聘理由书 | 公司公章 |
| 身元保证书 | 公司公章 + **公司法人章** |
| 查证申请人名簿 | 公司公章 |
| 滞在日程表 | 公司公章 |

> ⚠️ **注意**：也存在团组邀请函仅身元保证书盖章，其余三份文件不盖章递交至领馆的情况，一般不影响签证出签。

### 加急处理
- 日本签证**无需预约时间**
- 如出访时间紧张，签证材料可由市外办与省外办沟通选择**自送自取**
  - 需早上 **8:30前** 送至：
  - **江苏省签证服务中心上海部**
  - 地址：上海市黄浦区打浦路15号中港汇3001室

## 6. 签证材料注意事项 — 申根国家（以德国为例）

### 申根签证基本原则
- 出访两个及以上申根国家 → 以**停留时间长的为主签国**
- 停留时间相同 → 以**首入申根国**作为主签国

### 一、邀请函
| 要求 | 说明 |
|:---|:---|
| 发函单位 | 需是**外资企业** |
| 邀请人 | 外籍 / 已入外籍的华人 / 外企聘请的华人高管 |
| 出访时间 | 必须与**任务批件一致** |
| 内容格式 | 包含"江苏签证"德国签证邀请函要素，符合发函单位国家的行文格式（省外办要求） |

### 二、派遣函
| 要求 | 说明 |
|:---|:---|
| 出行时间 | 与邀请函时间一致 |
| 派遣身份 | **均为对外职务**，不可用党内身份申请出访（涉及职务的材料统一为对外身份） |
| 证明人 | 一般为出行人的上级（或同级） |
| 单位公章 | 盖**五角星章**，不可以是国徽章或镰刀斧头章 |

## 7. 江苏省外办签证办理重要提醒

> 以下为省外办整理的8条重要提醒，建议收藏保存。

### 提醒 1：以"因公签证要求"为准
- 通过 **[江苏省出国签证服务中心](http://www.visajs.com/)** → "各国因公签证要求"栏目查询
- 使领馆网站上的签证要求针对**因私签证**，和因公的材料要求**不一定相同**

### 提醒 2：组团之初提前对照签证要求
- 重点查看请示内容中是否有需要提前注意或规避的内容
  - **邀请函**：有的国家要求注明具体日程，有的要求提供护照号码 — 和外方对接时做好心中有数，避免重复联系
  - **组织代码证**：若有团员真实身份敏感不适合对外，或单位无法提供组织机构代码证，需考虑**更换对外身份**

### 提醒 3：不得随意变更出访行程
- 违反**外事出访纪律**
- 若出访国为互免、电子签等需开具出境证明的国家，改变行程将导致出境证明**无法使用**

### 提醒 4：申根国材料特别注意

| # | 注意事项 |
|:-:|:---|
| ① | 签证时间指的是**抵离出访国（境）的时间**，而非我国国境。所有材料（签证表、邀请函、派遣函、行程单、机票订单、酒店订单）中的时间节点务必**完全一致** |
| ② | 邀请函行文格式：致函对象可以是组团单位，也可以是被邀请人（一般为**团长**），**不要直接致函给使领馆** |
| ③ | 若同时出访申根两国，邀请函格式和内容**务必不能雷同** |
| ④ | 名单和行程定好尽早联系预约**录指纹时间**。如团员在59个月内录过申根指纹的，在"是否录指纹"栏填**"否"**（特定国家指纹不互通的除外） |
| ⑤ | **希腊、瑞士**：需提供一套电子扫描材料 |
| ⑥ | **波兰**：材料均为团组**自送领馆**。自送前到外办领取护照和照会；取签可选团组成员本人自取或沪办代取 |
| ⑦ | 签证材料须提前进馆，在约定录指纹日期前**至少2个工作日**到外办包签证材料 |

### 提醒 5：需在省一体化系统上报签证材料的国家
- **美国、加拿大、澳大利亚、新西兰、瑞士** — 需在省一体化系统"签证办理"模块同步上报签证材料
- **美国**：自行在网站上预约面谈时间和地点，面谈前准备好相关材料并提前到外办领护照

### 提醒 6：材料不得盖国徽章
- 团组准备的所有材料（外办出具照会除外）**一律不得盖国徽章**

### 提醒 7：电子签国家需领取出境证明
- **韩国、澳大利亚、新西兰**等需办理电子签的国家
- 出访前除护照外还须**领取出境证明**（外办开具）
- 按要求自行下载**电子签确认书/电子签证信**
- 出访时须**一并携带**

### 提醒 8：拿到护照第一时间检查签证页
- **领馆签证可能有错漏**（个人信息、签证有效期等）
- 发现问题**第一时间提出**，在出发前完成修正

### 8.1 德国 🇩🇪

| 项目 | 内容 |
|:---|:---|
| **送签地点** | 德国驻上海总领事馆 |
| **所需时间** | 短期（90天内）：至少预留 **7个工作日** 在馆审核。建议提前三周提交，旺季提前六周 |
| **长期签证** | 90天以上需提前两个月左右申请 |

**材料要点**：
1. **中文个人资料表**（供填写签证申请表）
2. **护照原件** — 有效期半年以上+本人签名；提供护照信息页复印件1份；如录过指纹，提供最近59个月内有效申根签证复印件
3. **邀请信原件** — 需邀请单位信笺纸打印，邀请人亲笔签名（电子签名不能申请签证）；写明访问目的、时间、行程、被邀请人姓名职务；如已持有有效护照，须注明护照号码；费用负担须写明具体单位；如顺访其他申根国家，需提供前往国邀请信原件
4. **签证申请表**（A3正反面打印，申请人亲笔签名）
5. **照片** — 1张近3个月白底小二寸
6. **派遣函**（单位信笺纸打印，盖章）
7. **行程表**（英文）
8. **声明表AB面**（正反面打印）
9. **指纹通知单** — 无论是否需要录指纹都需预约名额

### 8.3 法国 🇫🇷

| 项目 | 内容 |
|:---|:---|
| **送签地点** | 法兰西共和国驻上海总领事馆（公务普通护照：法国**南京签证中心**递交材料并录指纹） |
| **所需时间** | 短期：在馆审核 **7个工作日以上**；长期：提前至少20个工作日 |

**重要通知**（自2024年5月2日起）：
- **公务普通护照**持有人在**法国南京签证中心**递交纸质材料并采集指纹

- 同团组中既有公务护照又有公务普通护照 → 全团均在法国驻上海总领事馆递交材料并采集指纹

**材料要点**：
1. **中文个人资料表**
2. **护照原件**
3. **签证申请表**
4. **邀请信**
5. **派遣函**
6. **照片要求**（参见专用说明）
7. **公务普通护照申请信息表**（仅限在南京/上海法国签证中心录指纹团组填写）

### 8.4 瑞士 🇨🇭

| 项目 | 内容 |
|:---|:---|
| **互免协定** | 持有效外交护照，任意连续180日内停留不超过90天免签，需开具出境证明 |
| **所需时间** | 公务护照：进馆后**10个工作日以上**；公务普通：进馆后**5个工作日以上**；混合团按10个工作日 |

**材料要点**（不论有没有申根指纹，均需预约进馆名额）：
- **公务护照申请**：只需签证表、邀请函原件、护照复印件
- **公务普通护照 — 邀请方为国际组织**（38家国际组织名单参见外办）：签证表、邀请函、保单、护照复印件
- **公务普通护照 — 邀请方非国际组织**：签证表、邀请函、组织机构代码/营业执照复印件、派遣函、保单、护照复印件
- 材料每人一套，单面打印，不能订书针，按序号排放，夹在各自护照中
- 照片要求：35mm×40mm，近6个月白底彩照

5. **邀请信原件**
6. **派遣函**
