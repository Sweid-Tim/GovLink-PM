## 路线总览

```
上海 → 伦敦(3晚) → 巴黎(1晚) → 阿维尼翁(1晚) → 马赛 → 上海
```

| 项目 | 内容 |
|:---|:---|
| **天数** | 7天（1月25日 周六 — 1月31日 周六） |
| **涉及国家** | 英国、法国 |
| **公务场次** | 4场（斯特拉特福德学院 / 伦敦足球学校 / 至点影业 / 让·维拉尔戏剧文化中心） |
| **交通方式** | 飞机（东航/英航/东航）+ 欧铁（巴黎→阿维尼翁）+ 全程用车 |
| **住宿** | 伦敦3晚 → 巴黎1晚 → 阿维尼翁1晚 |

## 方案分析

### 亮点
| # | 亮点 | 说明 |
|:-:|:---|:---|
| 1 | **公务节奏合理** | 4场公务分散在D2/D3/D4/D5，每场有充分的交流时间，不与游览冲突 |
| 2 | **凌晨到达预案** | D1凌晨6:30到伦敦，已预付一晚房费让团组先入住休整，很人性化 |
| 3 | **交通组合丰富** | 中英段东航直飞、英法段英航、法国内段欧铁，体验多样 |
| 4 | **公务与文旅平衡** | 每天既有公务也有文化体验，不枯燥 |
| 5 | **细节到位** | 标注了官导、车程时间、参与人员名单及职务、议程时间线 |

### 注意事项
| # | 注意点 |
|:-:|:---|
| 1 | D6车程较长：阿维尼翁→埃兹(3.5h)→尼斯(30min)→马赛(2.5h)，当天合计约6.5h车程，团组较疲惫 |
| 2 | D3行程偏满：大英博物馆（含官导约2h）+ LFS公务 + 国家美术馆 + 科文特花园，一天内容较多 |
| 3 | D4巴黎段紧凑：15:10到酒店 → 16:00拜访Solstice Films → 之后还要走3-4个点+塞纳河游船 |
