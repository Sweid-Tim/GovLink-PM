# 0801-2（实际）出访活动日程表【越南】

> 本团组为日本+越南联程团组，越南段为第4-7天。前段日本行程详见 [[raw/sources/项目部/行程案例/日本行程]]。

---

## Day 4 | 8月7日（星期四）| 东京 → 河内

| 时间 | 活动 |
|:---|:---|
| 16:35-19:45 | ✈️ 东京→河内（航班 **VN385**，飞行时长约3h10min） |
| 19:45- | 驱车前往酒店，办理入住后休整（27km，约1h） |
| 晚上（可选） | 河内老街夜市、啤酒街 |

**住宿**：河内安纳托尔酒店（Anatole Hotel Hanoi）
地址：26-28-30 P. Nhà Chung, Street, 还剑郡, 河内

**导游注意事项**：
- 提前与酒店沟通，是否可凭护照照片提前拿房卡；如不可，导游统一办理入住，城市税/押金先行垫付
- 提前到机场等待，举牌接"QD"
- 接机携带打火机
- 确认客人是否需要夜宵，注意饮食卫生（客人之前吃坏过肚子）
- 每晚与客人确认次日行程及用餐
- 车内备好水和够5人使用的雨伞

## Day 5 | 8月8日（星期五）| 河内 → 胡志明

| 时间 | 活动 |
|:---|:---|
| 06:30-09:30 | 酒店早餐，用餐后退房 |
| 08:15-08:30 | 驱车前往公务地点（1.5km，15min） |
| 08:30-10:00 | **拜访越南计划投资部** |
| | 地址：No. 6-8 Phan Huy Chú, Hà Nội (Ministry of Finance) |
| 10:00-11:30 | 驱车前往西湖附近用餐，途经升龙皇城、镇国寺 |
| 11:30-12:40 | 午餐（推荐：西湖湖畔 Phương Nguyên 餐厅，已预定11:30包间） |
| 12:40-13:30 | 驱车前往公务地点（13km，50min） |
| 13:30-15:00 | **拜访 Hanoi Plastics JSC（河内塑料股份公司）** |
| | 地址：Group 12, Phuc Loi Ward, Hanoi, Vietnam |
| 15:00-16:00 | 驱车前往机场候机（32km，1h），机场简餐 |
| 19:00-21:15 | ✈️ 河内→胡志明（航班 **VN219**，飞行时长2h15min） |

**住宿**：胡志明西贡大酒店
地址：8 Dong Khoi Street, Ben Nghe ward, 第一区, 胡志明市

**导游注意事项**：
- 请联系翻译，翻译随车，车上沟通公务拜访拟交流内容
- 用餐途中做景点讲解，如客人有进景点需求，根据用餐时间沟通确认
- 协助点餐并垫付餐费（正餐人均约200元）
- 每次公务拜访前确认时间和地点，尤其是停车位
- 公务拜访协助拍照
- 送机后给客人充足越南盾用于机场点餐
- 客人入关后在机场停留30分钟
