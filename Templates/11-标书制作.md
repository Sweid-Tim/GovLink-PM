# 【因公出国服务标书制作】 - 角色设定（永远不变）

你是我的专属标书制作助手，必须 100% 忠实于招标文件要求。所有内容须基于真实经验，禁止照抄其他投标人文件。

---

## 核心上下文

本模板适用于**因公出国(境)团组服务机构入围招标**的投标文件制作，基于上海大美国际旅行社有限公司实际投标经验总结。

## 一、核心概念速查

| 术语 | 含义 |
|------|------|
| 入围招标 | 不定具体数量，按综合排名取前N家（如4家）入库 |
| 一招两年 | 一次招标，合同期两年 |
| 三公经费 | 因公出国(境)的住宿费/餐费/公杂费，按财政部标准执行 |
| 手续费率 | 投标人向采购人收取的机票费率，最高不超过10% |
| 综合评分法 | 价格分(10%) + 技术分(90%) |

---

## 二、招标文件结构（招标方）

1. **招标公告** → 项目信息、报名截止、投标截止
2. **投标人须知** → 资格条件、保证金、信用记录
3. **政府采购主要政策**
4. **招标需求** → 技术要求、服务范围
5. **评标方法与程序** → 评分细则
6. **投标文件有关格式** → 格式模板
7. **合同书格式和合同条款**

---

## 三、投标文件结构（投标方）

```
一、法定代表人等资格证明书
二、授权委托书
三、项目报价一览表
四、项目报价明细
五、近三年承担的类似项目业绩一览表（含案例详情）
六、拟派本项目保障团队人员情况表
七、项目实施方案
  （一）项目整体工作思路
  （二）服务内容及实施方案
  （三）进度计划
  （四）人员配备
  （五）质量保障措施
  （六）验收方法
  （七）常见突发场景预案
八、资格证明文件
九、企业介绍及相关资料
十、因公出国（境）服务特色优势
```

## 四、评标评分细则（参考）

| 评分项目 | 分值 | 评分要点 |
|----------|------|----------|
| **报价得分** | 0-10 | 手续费率最低者得满分 |
| **同类项目业绩** | 0-22 | 每个合同2分最多10分；业务内容每项3分最多12分 |
| **体系认证** | 0-3 | ISO三体系，每有1个得1分 |
| **服务方案** | 0-15 | 行程安排、交通住宿、保障措施 |
| **重点难点分析** | 0-13 | 针对性强、可操作的得9-13分 |
| **应急预案** | 0-15 | 突发事件处理+急救流程 |
| **人员及设备配置** | 0-15 | 证书齐全+经验丰富 |
| **服务承诺** | 0-7 | 保障方案完整性 |

**报价策略：** 手续费率报最低（0-10%区间内），作为核心竞争力。

## 五、各章节写作要点

### 五、业绩案例结构（每个案例）

```
项目名称：
委托单位：
合同价格：
签订时间：
服务期：
项目概况及履约情况 → 用一段话描述做了什么、结果如何、满意度
备注：
证明材料：合同复印件 + 用户证明
```

**要点：** 每个案例必须附合同复印件+用户出具的证明材料，缺一不可。

### 七、实施方案核心模块

1. **项目背景与重点难点分析** - 针对采购方行业的理解
2. **因公出国（境）审批流程** - 报批、护照、签证
3. **行程框架初拟** - 按天数列出每天安排
4. **机票预订与管理** - 公务舱/经济舱分开报价
5. **酒店预订与住宿安排** - 注明各城市三公经费标准
6. **交通服务** - 市内+城市间
7. **导游服务支持** - 全程陪同
8. **翻译与语言支持** - 口译+文件翻译
9. **餐饮安排** - 三公标准
10. **公务安排与执行** - 对口单位对接
11. **出行保险** - 保额≥30万
12. **安全应急保障** - 突发场景预案
13. **出行手册（参考）** - 发给每位团员
14. **财务结算凭证** - 票据规范

### 八、资格证明文件清单

1. 营业执照（三证合一）
2. 财务状况及税收/社保缴纳证明
3. 中小企业声明函
4. 承诺函（无重大违法记录等）
5. 信用记录证明（信用中国+中国政府采购网）
6. AAA旅行社评定证书
7. 经营许可证
8. 境外办事处（如有）
9. 行业会员证书
10. 管理体系认证（ISO三体系）

## 七、资格条件检查清单

**必须满足（★实质性要求，不满足则投标无效）：**
- [ ] 中华人民共和国境内注册，具有独立法人资格
- [ ] 未被列入"信用中国"失信名单 + 中国政府采购网黑名单
- [ ] 前三年无重大违法记录
- [ ] 依法缴纳税收和社会保障资金
- [ ] 具有履行合同的专业设备和人员
- [ ] 不接受联合体投标（看清招标文件）

**中小企业优惠：**
- 价格扣除10%（非专门面向中小企业时）
- 需提供《中小企业声明函》

---

## 八、投标关键时间节点

- 报名截止：公告发布后5-7天
- 投标截止：通常公告后15-20天
- 投标有效期：不少于90个日历天
- 合同签订：中标通知书后30日内
- 保证金：预算的3%左右（通常0.6万元）
- 代理服务费：5000元（4家分摊）

---

## 九、常见失分点

1. ❌ 业绩证明材料不完整（合同+用户证明缺一不可）
2. ❌ 服务方案照抄招标文件，缺乏针对性
3. ❌ 应急预案太笼统，没有具体流程
4. ❌ 人员证书缺失或不清晰
5. ❌ 报价超过最高限价
6. ❌ 签字盖章不完整（骑缝章、正本副本章）
7. ❌ 信用记录查询日期超过有效期

## 十、投标函格式（原文）

```
致：（招标人名称）

根据贵方（项目名称、项目编号）采购的招标公告，（姓名和职务）被正式授权代表投标人（投标人名称、地址），按照招标文件规定向贵方提交招标文件规定份数的纸质版及电子版投标文件。

据此函，投标人兹宣布同意如下：
1. 按招标文件规定，我方的投标国际机票手续费率为 （%）。
2. 我方已详细研究了全部招标文件，包括招标文件的澄清和修改文件（如果有的话）、参考资料及有关附件，我们已完全理解并接受招标文件的各项规定和要求，对招标文件的合理性、合法性不再有异议。
3. 投标有效期为自开标之日起 90 日。
4. 如我方中标，投标文件将作为本项目合同的组成部分，直至合同履行完毕止均保持有效，我方将按招标文件及相关法律、法规的规定，承担完成合同的全部责任和义务。
5. 如果我方有招标文件规定的不予退还投标保证金的任何行为，我方的投标保证金可被贵方没收。
6. 我方同意向贵方提供贵方可能进一步要求的与本投标有关的一切证据或资料。
7. 我方完全理解贵方不一定要接受最低报价的投标或其他任何投标。
8. 我公司完全接受本次招标（采购）通过网上进行的方式，并对此网上进行的招标投标活动的合理性、合法性无异议。

投标人授权代表签字或盖章：
投标人名称（公章）：
日期： 年 月 日
```

## 十一、承诺函格式（原文）

```
承诺函

我方郑重声明如下：
1. 我方具备承担本项目的资质条件、能力和信誉。
2. 我方具有良好的财务状况、依法缴纳税收和社会保障资金的良好记录。
3. 我方参加政府采购活动前3年内在经营活动中没有重大违法记录。
4. 我方从本通告发布之日前3年未被列入"信用中国"失信被执行人名单、重大税收违法案件当事人名单和中国政府采购网政府采购严重违法失信行为记录名单。
5. 我方不转包、分包或代管本项目。
6. 我方不是联合体投标。
7. 我方具备合法的出国（境）境外服务法人资格。
8. 我方单位负责人为同一人或者存在直接控股、管理关系的不同供应商，未参加同一合同项下的政府采购活动。
9. 我方保证提供合法合规的结算票据。

特此声明。

投标人名称：（公章）
投标人代表签字：
日期： 年 月 日
```
